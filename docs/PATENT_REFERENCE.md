# Patent Reference Checklist

OpenEncoder may use the public-safe phrase "patent pending". Do not include the submitted filing PDF, OCR text, attorney packet, private addresses, private e-mail addresses, claim text, or non-public application details in this repository.

## Current Public Repo Wording

```text
Patent pending. Public application details are intentionally omitted until a publication record or maintainer-approved citation is available.
```

## Needed For Full Citation

- Official public record link from the patent office or publication database.
- Publication number or patent number once available.
- Jurisdiction and application type that may be disclosed publicly.
- Official title exactly as it appears in the public record.
- Filing date and publication or grant date once publicly available.
- Applicant/assignee and inventor names only if already public or approved for publication.
- Counsel-approved wording for README, package metadata, website copy, LICENSE, and NOTICE.
- Feature scope: which OpenEncoder features may be tied to the filing and which must not be implied.
- License impact: patent grant, defensive termination, commercial carveout, or NOTICE language.

## Publication Rule

Until an official publication record exists or maintainers approve exact public wording, the repo should say only "patent pending" and should not publish application numbers, private filing forms, private contact details, or claim language.
